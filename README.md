# Main Fuctions
### ✅리서치 참여
#### 사용자 정보를 바탕으로 자체 스크리닝을 거친 참여가능한 리서치에만 알림을 통해 참여하세요!
### ✅리서치 의뢰
#### 인터뷰/테스터/설문조사, 어떤 리서치든 의뢰가능합니다. 만드신 설문링크를 그대로 올려도 좋아요!
### ✅합당한 보상
#### 리서치에 성실히 참여한 만큼 포인트가 지급되며, 이는 곧 개인의 금전적인 보상으로 이어집니다!
